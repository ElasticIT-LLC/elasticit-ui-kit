# @elasticit-llc/ui-kit

Optional React component library for ElasticIT portal apps.

## Installation

```bash
npm install @elasticit-llc/ui-kit recharts
```

Requires `.npmrc` with:
```
@elasticit-llc:registry=https://npm.pkg.github.com
```

## Usage

```tsx
import { DataTable, KPICard, StatusBadge, Chart, exportCsv } from '@elasticit-llc/ui-kit'
```

## Development

```bash
npm install
npm run build
npm run test
```
